Properticons

Properticons is an icon font for Real Estate sites, themes, and plugins.

Show instructions

Photoshop Usage

Use the .OTF version of the font for Photoshop mockups, the web-font versions won't work. For most accurate results, pick the "Sharp" font smoothing.

CSS Usage

Link the stylesheet from our CDN (provided by MaxCDN):

<link rel="stylesheet" type="text/css" href="http://maxcdn.properticons.com/1.0.0/css/properticons.css">

Now add the icons using the :before selector. You can insert the Bed icon like this:

.myicon:before { }
